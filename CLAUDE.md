## Project Overview

This is a simple static website for OwnTube Nordic AB (Org. Nr: 559517-7196), a Swedish consulting company specializing in React development, video streaming infrastructure, identity/authorization systems, container platforms, and backend development.

The repository contains:
- A corporate website with Swedish content describing the company's services
- Static site hosted on GitHub Pages at www.owntube.se
- Company logos and brand assets in assets/logos/ directory
- Minimal structure with README, CNAME, LICENSE, and brand assets

## Architecture

This is a basic static website repository with no build process, dependencies, or complex architecture:

- **README.md**: Swedish-language company information and service offerings
- **CNAME**: GitHub Pages custom domain configuration pointing to www.owntube.se
- **LICENSE**: Public domain license (Unlicense)
- **assets/logos/**: Company brand assets including SVG and PNG logos in various formats (square icons, rectangular logos, color variants)

## Development

Since this is a static site with no build process:
- No package.json, dependencies, or build commands
- No testing framework or linting tools configured
- Changes can be made directly to files and committed
- Site automatically deploys via GitHub Pages when pushed to main branch

**Language**: This project uses Swedish for commit messages and GitHub issues.
